# OpenDART + XBRL 자동 추출 검증

## 처리 흐름

1. OpenDART API 인증
2. 사업보고서 접수번호 자동 검색
3. 2023~2025 연결재무제표 자동 수집
4. 주요 재무계정 표준화
5. XBRL 원문 자동 다운로드
6. 연결 기준 전체 영업권 기말잔액 자동 추출
7. 11개 필수 계정 CSV 생성
8. 결측값 검증
9. Red Flag Agent 실행
10. JSON 및 Markdown 보고서 생성

## 최종 검증 결과

- 기업: 하이브
- 분석기간: 2023~2025
- 필수 계정: 11개
- 결측값: 0
- Agent 종료 코드: 0
- Red Flag: 7개
- Monitoring Signal: 3개

## 영업권 자동 추출

영업권은 기본 재무제표 API에서 무형자산과 합산되어 표시될 수 있어,
사업보고서 XBRL 원문에서 다음 조건을 만족하는 값을 자동 선택했다.

- 태그명: Goodwill
- 연결 기준: ConsolidatedMember
- 별도재무제표 및 세부 CGU 차원 제외
- 해당 사업연도 말 기준
- 단위: KRW

## 주의사항

식별된 Red Flag는 오류나 부정의 존재를 의미하는 것이 아니라, 기업 및 환경에 대한 이해와 중요왜곡표시위험 평가를 위해 추가적인 질문과 검토가 필요한 영역을 의미한다.
